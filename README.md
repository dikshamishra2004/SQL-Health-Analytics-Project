# SQL Health & Lifestyle Analytics Project

This project uses **SQL (PostgreSQL)** and **Tableau** to analyze a synthetic dataset of individuals’ health and lifestyle patterns. It includes over 20 SQL queries and a fully visualized dashboard with key health insights.

## Objective

To analyze health and lifestyle data to uncover trends and correlations related to:

- BMI and obesity
- Smoking and chronic disease risk
- Stress vs Sleep patterns
- Alcohol consumption and lifestyle habits

## Dataset Summary

- **File:** `synthetic_health_lifestyle.csv`
- **Columns:**
  - `age`, `gender`, `height_cm`, `weight_kg`, `bmi`
  - `smoker`, `exercise_freq`, `diet_quality`, `alcohol_consumption`
  - `chronic_disease`, `stress_level`, `sleep_hours`

## Tools Used

- **PostgreSQL** for SQL analysis
- **DBeaver** as SQL IDE
- **Tableau** for data visualization and dashboarding

## Tableau Dashboard

The interactive dashboard highlights key metrics and visual insights:

- **KPI Cards**:
  - Total Individuals
  - Obesity %
  - Average Sleep Hours
  - Smokers %
- **Visualizations**:
  - Chronic Disease by Gender
  - Stress vs Sleep Line Chart
  - BMI Distribution
  - Alcohol vs Chronic Disease Heatmap

## How to View the Dashboard

- Download and open the `Health Dashboard.twbx` file in Tableau Desktop
